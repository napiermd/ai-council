# Example: Class Project Review

## When to use

Use this when you are trying to decide what project to do, how broad to make
it, or whether your current project framing is actually defensible.

## Example prompt

```text
Use the AI Council with the Stanford Academic Pack.

Project idea:
<what I want to build or study>

Current framing:
<how I am pitching it now>

Constraints:
- class timeline
- team size
- available data
- expected deliverable

What I want:
- tell me if this is too broad or too thin
- identify the strongest angle
- tell me what kind of evidence I can realistically produce
- give me the next scoping move
```
