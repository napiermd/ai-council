# Example: Clinical AI Ask Nigam

## When to use

Use this when the question touches healthcare data, clinical workflow, hospital
operations, or whether a clinical AI idea is operationally real.

## Example prompt

```text
Ask Nigam what he thinks about this clinical AI idea.

Context:
I am evaluating a clinical AI workflow that would help emergency physicians
generate structured documentation and decision support from chart context and
clinical notes inside the actual care workflow.

Idea:
Use an LLM-based system to organize clinical information, reduce documentation
burden, and surface useful synthesis without creating more alert fatigue.

Decision:
Should I treat this as a real health-system product direction worth building,
or is it more likely to fail on data quality, workflow friction, or adoption?

I want the answer anchored on:
- data realism
- workflow fit
- operational value
- deployment friction

Then tell me which other council voices, if any, should be added.
```

## Recommended voices

- Nigam Shah first
- Jonathan H. Chen if bedside workflow is central
- Chip Huyen if system reliability and deployment are major concerns
- Fei-Fei Li if institutional or human-centered framing matters

## Why start with Nigam

Nigam is the right first filter when the main question is whether an idea
survives real health-system data, workflow, and organizational conditions.
