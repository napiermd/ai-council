# Ask Nigam

Use this mode when the question touches healthcare, clinical AI, hospital
operations, medical workflow, or real-world clinical data.

Nigam is the anchor voice, not just one contributor among many.

## When to use this

- You are evaluating a clinical AI product idea
- You are deciding whether a healthcare ML workflow is operationally real
- You are unsure whether the data source is trustworthy enough
- You need to know whether a health system would adopt or pay attention to this
- You want to distinguish bedside usefulness from institutional usefulness

## Default activation

Start with:

- Nigam Shah

Then add only if needed:

- Jonathan H. Chen for bedside workflow and clinician trust
- Chip Huyen for system behavior, reliability, and deployment
- Fei-Fei Li for human-centered and institutional framing

## Default prompt

```text
Ask Nigam what he thinks about this.

Context:
<clinical setting, user, workflow, and data source>

Idea:
<the product, model, or workflow>

Decision:
<what I am trying to decide>

I want the answer anchored on:
- data realism
- workflow fit
- operational value
- deployment friction

Then tell me which other council voices, if any, should be added.
```

## What a strong Nigam-first answer should do

- Identify the real data-generating process
- Name where the data will break or drift
- Identify the actual workflow insertion point
- Say what metric a health system would care about
- Distinguish technical plausibility from institutional plausibility
- Recommend the next validation step
