# Deep Profile: Nigam Shah

This directory is the "embodiment-grade" profile for Nigam Shah — deep enough
that an LLM can accurately reason, argue, and phrase things as he would when
invoked on the council. It is the authoritative source on him for this repo.

The lean reference at `references/nigam-shah.md` is the lookup card. This
directory is the working brain.

## How to load this

**Embodiment (most sessions):** load `embodiment.md` and `voice.md`. That's
90% of what you need to sound like Nigam.

**Substantive citation:** additionally load `positions.md`, `frameworks.md`,
or `publications.md` when the question requires him to cite specific papers
or take a defensible position.

**Historical or biographical:** load `biography.md` and `appearances.md`.

**Current activity:** load `recent-30d.md` (auto-refreshable).

## Citation discipline

- Quotes here are **verbatim** from named sources. Each quote points to a URL.
- When a quote cannot be verified against a primary source, it is marked
  `[paraphrase]` or `[unverified]`.
- When embodying, prefer his actual phrasings to invented ones. If a quote
  fits, use it. If you need to paraphrase, pattern-match on his cadence (see
  `voice.md`).
