# Biography: Simon Willison

## Name

Simon Willison. `@simonw` on GitHub, X, Mastodon, Bluesky. British,
based in Half Moon Bay, California.

## Education

- **B.Sc. Computer Science**, University of Bath (2005)

## Career arc (verified)

| Years | Role |
|---|---|
| 2017–present | Independent OSS developer. **Datasette Inc.** (founder; running Datasette Cloud). Daily blogging at simonwillison.net. 641+ GitHub Sponsors. PSF Board since 2022 |
| 2019–2020 | **JSK Fellowship at Stanford** (journalism) — runway to focus on Datasette full-time |
| 2013–2017 | **Engineering Director, Eventbrite** (after Lanyrd acquisition; relocated to San Francisco) |
| 2010–2013 | **Co-founder, Lanyrd** (conference directory). With Natalie Downe (now his wife). Y Combinator early 2011. $1.4M seed September 2011. Acquired by Eventbrite, 2013 |

| 2005–2008 | **Yahoo** (Technology Development) — Fire Eagle geolocation |
| 2003–2004 | **Industrial placement, Lawrence Journal-World** (Kansas). The year Django was born. He, Adrian Holovaty, Jacob Kaplan-Moss, Wilson Miner built what became Django, released 2005 |
| 2000 | Webmaster at Gameplay; part-time at Incutio while at Bath |

## Key projects (time-ordered)

- **Django (2005)** — co-created with Adrian Holovaty. The Python web
  framework. 20+ years of use, still dominant.
- **Lanyrd (2010-2013)** — conference discovery. Built on honeymoon.
  Sold to Eventbrite.
- **Datasette (2017-present)** — his flagship OSS project. 11K stars.
  "A tool for exploring and publishing data." 44 tools + 154 plugins
  in ecosystem.
- **sqlite-utils (2018-present)** — 2K stars. SQLite command-line utility.
- **shot-scraper (2022-present)** — 2.3K stars. Playwright-based
  screenshot automation.
- **`llm` CLI (2023-present)** — 11.6K stars. AI-era flagship. Unified
  CLI + Python library for every major model + local.
- **Pelican-on-a-bicycle benchmark (October 2024)** — his visual
  capability test. Repo: github.com/simonw/pelican-bicycle.
- **tools.simonwillison.net (2024-present)** — 80+ single-purpose
  HTML+JS tools, mostly LLM-assisted. "Prompt-driven development with
  very low stakes."
